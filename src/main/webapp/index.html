Comet Send Test (warmup)

Setup/Description

You will need two computers. Other to run Lift server and client for the server. Laptop 2 connects to the Lift server, sends messages and closes the lid.

Test

  1. Laptop 2 sends a message "Hi all!" => All clients connected to the Lift server shows the chat message
  2. Laptop 2 closes laptop lid
  3. Laptop 2 opens laptop lid after 4 minutes
  4. Laptop 2 sends new message "delayed"

Result

  • Other clients connected to Lift server shows "delayed" message immediately
  • Laptop 2 shows sent "delayed" message in about 2 minutes
